    annotation Testify::Test::Tags

    Overview

    Tags can be used to group tests and specs, allowing to only run a subset of tests and/or specs when providing a --tag argument to the spec runner.

    Can be tagged:

    • classes inherited from the Test class (equivalent of describe block);
    • and all test_ methods (equivalent of it block);
    • and of course the blocks (with the tags argument): describe, context and it.
    See Tagging Specs in the stdlib.
